Streamlit Cloud 在线应用

自动化订单清洗与映射系统

这是一个面向 Excel 订单数据的自动清洗工具，支持批量处理、字段映射、手机号归属地识别与结果导出。点击下方按钮即可在线使用，无需安装任何软件。

🚀 打开在线应用 查看源代码

项目特点

  • 上传 Excel 文件后自动清洗订单数据
  • 完成年级、渠道、推广计划名称的自动映射
  • 手机号归属地自动识别并分类
  • 生成可直接上传的清洗后 Excel 文件

使用方法

  • 点击上方"打开在线应用"进入 Streamlit 应用页面
  • 上传从系统导出的订单 Excel 表格
  • 点击"开始执行全自动清洗 SOP"按钮
  • 处理完成后点击下载清洗结果文件
